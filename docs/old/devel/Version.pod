=head3 Overview

All the sources for the =PROJECT= project are kept in a CVS version control
repository on soundwave.mcilink.com. This repository is currently being
maintained by Paul Houghton <paul.houghton@wcom.com>, please contact
him for remote write access or any problems you encounter.  For read
only access to the repository you can use the B<cvsguest> id.

=head3 Getting The Sources from CVS

First, set your CVSROOT environment variable to:

  export CVSROOT=:pserver:cvsguest@soundwave.mcilink.com:/Tools/src/CvsRepository

Then follow the instructions in
http://soundwave.mcilink.com/Tools/VersionControl/CvsQuickRef.html to
checkout a copy of the sources using the project name of
B<lib=PROJECT=-=VER_MAJOR=>.
