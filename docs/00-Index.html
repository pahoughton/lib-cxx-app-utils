StlUtils C++ Library

June 4, 2000
Paul A. Houghton (paul.houghton@wcom.com)

StlUtils is a library generic of C++ utility classes and functions. These classes are useful for any development project.

Each class in this library has a header file with the same name as the class (i.e. the Compress class'es header is Compress.hh). The primary documentation for the classes is in the header file after the class declaration.

Warning: This documentation is still under development there may be some errors and omissions. Please provide feedback when you find a problem with it.
